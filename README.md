ส่วนคำนวนและตรรกะ
<img width="443" height="457" alt="Screenshot 2026-02-15 023118" src="https://github.com/user-attachments/assets/0ed37589-2d97-4e53-b40a-b071ec227177" />

โดยการทำงานหลักๆ คือ ในตารางคำนวนจะไปดึงค่า input จากตาราง input มา และจากนั้นจะตรวจสอบดูว่า
input นั้นเป็นค่าอะไร(X, O) หากเป็น X จะกำหนดค่าเป็น 1 ลงในช่องๆนั้นของตารางคำนวน ในทำนองเดียวกัน
หากเป็น O จะกำหนดค่าเป็น -1 

จากนั้นจะคำนวนผลรวมในแต่ละ หลัก(col) แถว(row) และ แนวทะแยง(diagonal) หากเป็น 3 คือ X ชนะ
ก็จะกำหนดค่าเป็น 1 ลงในช่องคำนวนผลรวมนั้น และเช่นเดียวกัน หากเป็น -3 คือ O ชนะ และจะกำหนดค่าเป็น -1

ส่วนการแสดงผลลัพท์ จะหาผลรวมจากช่องคำนวนผลรวมทั้งหมดหาก มีค่า 1 หรือ -1 จะแสดงผลเป็น "X Won" 
และ "O Won" ตามลำดับ เนื่องจากการที่จะมีผลรวมเป็น 3 หรือ -3 ได้นั้นหมายความว่าจะต้องมี X หรือ O เรียงต่อกัน 3 ตัว
